## 2. 增强定投策略

本策略采用**增强定投法**：不依赖均线、估值或宏观预测，只关注**上一次定投时的价格**。

### 2.1 核心参数

- **定投日**：每月第三个周五（遇节假日顺延至下一交易日）
- **基准金额**：3000元（可调整）
- **调节比例**：±20%

### 2.2 执行逻辑

在每个定投日，对比**现价**与**上次定投价格**：

| 条件 | 操作 | 公式 |
| :--- | :--- | :--- |
| 现价 > 上次价（涨了） | 少买 | 基准金额 × 0.8 |
| 现价 < 上次价（跌了） | 多买 | 基准金额 × 1.2 |
| 现价 = 上次价 或 首次 | 按基准买 | 基准金额 |

**记忆单元**：每次定投后，必须将当日成交价记录为新的"上次定投价格"，供下月对比。

**更新规则**：仅在实际发生买入时更新该标的的"上次定投价格"；若某标的当月未买入（如停牌、缺数据），则不更新，下月继续用原值对比。

### 2.3 新增标的

中途加入新标的时，首次建仓按**基准金额**执行，之后按上述规则执行。

## 3. 止盈与止损

### 3.1 买入端（被动风控）

定投阶段，风险控制内化于买入逻辑中，**无需主动卖出**：

- **被动止盈**：市场上涨时，越买越少，减少高位筹码堆积。
- **被动止损**：市场下跌时，越跌越买，极速摊薄成本，回本更快。

### 3.2 卖出端（主动止盈）

**定投阶段不做主动止盈**。积累期只依靠买入端逻辑（涨少买、跌多买）控制风险，不因浮盈或回撤而主动卖出。

**兑现时机**：等真正需要用钱时（如退休、买房、子女教育等）再分批兑现，避免因回撤止盈而卖飞、错过后续涨幅。

### 3.3 止损

宽基指数、行业指数**不做主动止损**，依靠"越跌越买"摊薄成本。个股不适用本策略。

## 4. 仓位调整与再平衡

当各标的实际占比与目标比例不一致时，通过以下方式逐步靠拢目标，无需频繁卖出。

### 4.1 优先方式：用新增资金再平衡（每月执行）

不卖出现有持仓，在每月定投时做**分配倾斜**：

1. **算当前占比**：总资产 = 所有标的市值 + 现金；各标的占比 = 该标的市值 ÷ 总资产。
2. **对比目标**：与上文目标配置表对比，标出谁超配、谁低配。
3. **定投分配**：本月定投金额优先分配给**低配**的标的（低配越多，分到的金额越多）；**超配**的标的少分或本月不投。单标的内部仍按「涨少买、跌多买」计算该标的当次金额。

长期执行后，比例会逐步向目标靠拢。

### 4.2 可选：一次性主动调仓（偏离过大时）

若某标的偏离目标超过 **±5%～8%**（可自定阈值），可做一次主动调仓：

- **超配**：卖出部分持仓，使该标的占比回到目标附近。
- **低配**：用卖出所得 + 当月定投，优先买入低配标的。

调仓目的是恢复目标配置，不是止盈；卖出超配、买入低配，属于组合管理。

### 4.3 操作顺序小结

| 步骤 | 内容 |
| :--- | :--- |
| 1 | 计算总资产、各标的当前占比 |
| 2 | 与目标比例对比，列出超配/低配 |
| 3 | 每月定投：低配多投、超配少投或不投（结合涨少买跌多买） |
| 4 | 偏离过大时可选：卖出超配、买入低配，做一次调仓 |
